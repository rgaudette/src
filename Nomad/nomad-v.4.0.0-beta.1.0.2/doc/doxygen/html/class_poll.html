Generate the trial points (::start), launch evaluation (::run) and postprocecssing (::end).

void Poll::generateTrialPoints ( )
overridevirtual

Generate new points to evaluate.

The trial points are obtained by:

  • adding poll directions (::setPollDirections) to the frame center.
  • snaping points (and directions) to bounds.
  • projecting points on mesh.

Implements IterationUtils.

void Poll::setPollDirections ( std::list< Direction > &  directions) const
private

Helper generateTrialPoints.

  • A single direction on unit n-sphere is computer (::computeDirOnUnitSphere).
  • This direction is transformed into 2n directions on a uni n-sphere using the householder transformation.
  • The 2n directions are scaled and projected on the mesh.

/param directions The directions obtained for this poll – OUT.
