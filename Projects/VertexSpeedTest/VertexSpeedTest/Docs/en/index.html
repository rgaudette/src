This Handbook describes VertexSpeedTest Version 0.1

1. Introduction

This program explores the performance effects of different rendering techniques, memory types (system, AGP, video), and geometry techniques (triangles, triangle strips, trinage fans) for million vertex rendering problems. The different rendering techniques explored are:
  • immediate mode:
    • using glBegin/glEnd with glVertex*/glNormal*
    • using glBegin/glEnd with glVertex*v/glNormal*v
  • vertex array:
    • DrawArrays
    • ArrayElement in a loop
    • DrawElements
    • DrawRangeElements
    • InterleavedArrays
The different memory types and objects stored explored are:
  • memory type
    • system memory
    • AGP memory
    • video memory
  • other issues
    • VertexArrayRange
    • VertexArrayRange2
    • store indices as well as vertices in fast memory?
